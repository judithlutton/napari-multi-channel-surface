napari-multi-channel-surface documentation¶

napari-multi-channel-surface is a plugin for the open source bioimaging software napari, and is intended for use in napari. The goal of this plugin is to provide a means for reading, writing, and interacting with surfaces containing multiple color channels in napari. When reading, color channel data is imported into the napari.layers.Surface object metadata under the key point_data. Different channels can be used to color the surface in the napari viewer using the ‘Channel Select’ widget supplied with this plugin. Multi-channel surfaces are saved using the point_data metadata key to save each named channel.

Check out the Usage section for further information, including how to install the project.

Note

This project is under active development.
